                
1. Общие проблемы с UI
  * Зависания – перегрузить страницу.
  * Долгая отрисовка страниц Если перегрузка не помогает, проверьте общую загрузку системы.
  * Полная недоступность UI Переоткрыть браузер. Проверить доступность UI в другом браузере. Проверить доступность UI с другого IP.
2.	Отображение данных в аналитике
  * Занные приходят с запозданием Проверить по статусу БК, что есть данные от БК вообще. Сохраняется ли ситуация с задержками.
  * От одного или группы БК данные не приходят Убедившись, что данные перестали приходить – обратиться в службу поддержки.
  * Данные аналитики не соответствуют реальному видео. Зайти на записанное видео и найти там время, соответствующее времени наличия проблемы по аналитике.
3.	Работа оповещений
  * Оповещения не соответствуют видео Переоткрыть видео заново, проверить по аналитике время возникновения проблемы.
  * Отсутствуют оповещения, хотя видео говорит об обратном Проверить наличие настроенных оповещений на наблюдаемую ситуацию.
  * Висят активные, когда по видео видно, что не должны Проверить аналитику и убедиться в ложности оповещений. После чего их закрыть.
  * Не отображаются низкоприоритетные\высокоприоритетные оповещения Убедиться в том, что оповещения были созданы. 
4.	Невозможность создания или просмотра страниц меню Убедиться, что данному пользователю меню должны быть доступны. Выйти и заново зайти под пользователем.
  * Проверить роли и их доступы
  * Проверить обязательные поля
5.	Видео и аудио
  * Не работает записанное или стриминговое видео Проверить доступность видео по другой задаче. Проверить доступность данных по задаче
  * Качество видео не соответствует ранее наблюдаемому. Заново открыть видео, подождать.
  * Видео идёт без аудио Проверить аудио по другим каналам. Проверить аудио в записанных ранее видео, если возможно.
6.	Разное
  * На карте не видно нужного БК сделать zoom out/in и проверить доступность БK. Перегрузить страницу.
  * Исчез виджет с dashboard Перегрузить страницу и заново проверить доступность виджета. Пересоздать виджет.
